## Overview

This repository contains the full text of the **United Nations Universal Declaration of Human Rights (UDHR)** in plain text format. It is not a software project — there are no build systems, tests, or code.

## Files

- `unudhr.txt` — The full UDHR text: preamble followed by 30 articles. Article headers (e.g., "Article 1") appear on their own line followed by the article body.
